# 개미 전사

식량창고 N개에 대한 식량들이 주어딜 때 개미 전사들은 최소 한칸 이상씩 뛰어두며 식량을 약탈할 수 있다. 이때 일직선상에서 최대한 많은 식량을 얻기위한 최댓값을 구하시오

* 입력 예시
~~~
4
1 3 1 5
~~~

* 출력 예시 : 식량의 최댓값

## 문제 분석
* 일직선 상에 존재하는 값 즉 리스트에서 인접하지 않고 한 칸씩 뒤어가며 가져올 수 있는 최댓값 계산
* 왼쪽 부터 차례대로 작업하며 진행하며 이전 작업들은 고려하지 않는다.

---

## 소스코드 1

### 알고리즘
* 점화을 세운다. 이때 

### 점화식
$a_i = max(a_{i-1}, a_{i-2}+array[i])$

~~~
# 정수 N을 입력 받기
n = int(input())

# 모든 식량 정보 입력받기
array = list(map(int, input().split()))

# 앞서 계산된 결과를 저장하기 위한 DP 테이블 초기화
d = [0]*100

# 다이나믹 프로그래밍(Dynamic Programming) 진행 (보텀업)
d[0] = array[0]
d[1] = max(array[0], array[1])
for i in range(2, n):
    d[i] = max(d[i-1], d[i-2]+array[i])

# 계산된 결과 출력
print(d[n-1])
~~~
